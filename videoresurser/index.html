    Videoresurser för kursen¶

    Den här sidan samlar videor som används som komplement i studiecirkeln ”Studiecirkel – webb & AI”
    (webbutveckling, Git/GitHub, HTML/CSS och generativ AI).
    Information om videorna är hämtad november 2025.

    För en översikt över alla videor vi bedömt (även de som inte är med i kursen), se:
    videobank – bedömda videor

    Förslag på arbetssätt med videorna¶

    För varje video kan ni följa en enkel struktur:

    1. Inför visning
    2. Berätta kort varför ni visar just den här videon.

    3. Ge 1–2 frågor att ha i bakhuvudet, t.ex.:

      • ”Vad var nytt jämfört med hur vi nyss förklarade detta själva?”
      • ”Vad tycker du var mest oklart eller svårt?”

    4. Under visning

    5. Pausa gärna vid viktiga begrepp (t.ex. när Git-kommandon, HTML-taggar eller LLM förklaras första gången).

    6. Notera spontana frågor från deltagarna.

    7. Efter visning

    8. Låt deltagarna formulera med egna ord vad de såg.
    9. Koppla tillbaka till deltagarnas vardag och föreningens behov:
      • ”Hur skulle vi kunna använda det här i vårt eget projekt?”
      • ”Vilka risker eller farhågor väcker detta hos dig?”

    Videor som används i kursen¶

    Paket 1 – Gör din första hemsida¶

    1. Hur du kopplar källkod till GitHub och aktiverar GitHub Pages för HTML-sidor (på svenska)¶

    • Länk: https://www.youtube.com/watch?v=3hmz-nMtzTg
    • Längd: ca 10–15 minuter (uppskattning)
    • Svårighetsgrad: Nybörjare
    • Koppling till modul: Paket 1, modul 1.1 – Git, GitHub och GitHub Pages
    • Syfte i kursen:
      Visar steg för steg hur man:
    • kopplar lokal källkod till ett GitHub-repo
    • aktiverar GitHub Pages
    • får en HTML-sida att visas publikt.
      Används som ”titta tillsammans + pausa + labba” under träffen.

    2. Git – En introduktion på svenska (spellista)¶

    • Länk: https://www.youtube.com/playlist?list=PLwh73nXB3whgormDmX_skiiJfHfOa9Jgd
    • Längd: ca 1,5 timme totalt (7 delar)
    • Svårighetsgrad: Nybörjare–medel
    • Koppling till modul: Fördjupning till Paket 1, modul 1.1 och vidare in i Paket 2
    • Syfte i kursen:
      Mer heltäckande genomgång av Git:
    • grunder, installation
    • arbeta med filer och historik
    • grenar och sammanslagning
    • samarbeta med andra, push/pull, SSH-nycklar.
      Rekommenderas som självstudier mellan träffarna för deltagare som vill bli tryggare i Git.

    3. HTML och CSS – nybörjarguide på svenska (spellista)¶

    • Länk: https://youtube.com/playlist?list=PLuc3JXzcNTZSYxj8Q32SIbR7wt7fgi7of
    • Längd: ca 1 timme totalt (13 delar)
    • Svårighetsgrad: Nybörjare
    • Koppling till modul: Paket 1, modul 1.2 – HTML och CSS
    • Syfte i kursen:
      Huvudresurs för att lära sig HTML och CSS på svenska.
      Vi ser delar av spellistan tillsammans i två sittningar under samma träff, med paus för:
    • frågor
    • gemensam genomgång
    • praktiska övningar i deltagarnas egna sidor.

    Paket 4 – AI, LLM:er och agenter¶

    4. AI, Machine Learning, Deep Learning and Generative AI Explained (IBM Technology)¶

    • Länk: https://youtu.be/qYNweeDHiyU
    • Längd: ca 10 minuter
    • Svårighetsgrad: Nybörjare–medel
    • Koppling till modul: AI-delen, introduktion till generativ AI
    • Syfte i kursen:
      Översiktsvideo som knyter ihop begreppen:
      AI → Machine Learning → Deep Learning → Foundation Models → Generativ AI.
      Används efter att kursledaren först förklarat grunderna på svenska.

    5. How Large Language Models Work¶

    • Länk: https://youtu.be/5sLYAQS9sWQ
    • Längd: ca 8–12 minuter
    • Svårighetsgrad: Medel
    • Koppling till modul: AI-delen – Stora språkmodeller (LLM)
    • Syfte i kursen:
      Fördjupning om vad en LLM är, hur den tränas och används.
      Passar efter en enklare introduktion och gärna med paus på några ställen för frågor.

    6. AI Agents, Clearly Explained / AI Agents for Curious Beginners¶

    • Länk: https://youtu.be/FwOTs4UxQS4
    • Längd: ca 10 minuter
    • Svårighetsgrad: Medel–fortsättning
    • Koppling till modul: AI-delen – agenter och workflows
    • Syfte i kursen:
      Introducerar begreppet AI-agenter: från ”vanlig” LLM-chat → AI-workflows → mer autonoma agenter.
      Bra som framtidsspaning och underlag inför diskussion om hur vi kan bygga egna agenter i praktiken.

    7. The Limits of AI: Generative AI, NLP, AGI, & What’s Next? (IBM Technology)¶

    • Länk: https://youtu.be/rBlCOLfMYfw
    • Längd: ca 20 minuter (19:51)
    • Svårighetsgrad: Medel–fortsättning
    • Koppling till modul: AI-delen – etik, begränsningar och framtid
    • Syfte i kursen:
      Mer reflekterande video om vad AI faktiskt kan och inte kan.
      Tar upp:
    • begränsningar i generativ AI och NLP
    • AGI-diskussioner
    • risker, hallucinationer och hållbarhet.
      Passar bra som underlag för ett etik/förtroende-pass eller som frivillig fördjupning.
