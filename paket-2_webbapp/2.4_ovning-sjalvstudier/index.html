    2.4 Övning & självstudier – aktivitetslista som webbapp¶

    I den här övningen använder du det du jobbat med i 2.1–2.3 för att skapa en enkel aktivitetslista som webbapp.

    Målet är att sidan ska kännas mer som ett litet program än som en statisk sida.

    Du väljer själv ambitionsnivå – grund, mellan eller extra.

    Grundnivå (rekommenderad för alla)¶

    Målet på grundnivå är att ha:

    1. En lista som visas från JavaScript (array)
    2. Ett formulär som lägger till en aktivitet i listan
    3. Sidan uppdateras när du lägger till något

    Steg:

    • Utgå från koden i 2.2 (array + renderActivities() + formulär)
    • Anpassa till ditt eget sammanhang, t.ex.:
    • aktiviteter i föreningen
    • saker du vill göra i veckan
    • Lägg till minst 3–5 aktiviteter totalt (exempel + egna)

    Mellannivå¶

    Om du vill bygga vidare kan du lägga till minst en av följande funktioner:

    1. Visa antal aktiviteter
    2. Lägg till ett element, t.ex. <p id="count"></p>

    3. uppdatera texten varje gång renderActivities() körs

    4. Filtrera på plats eller typ

    5. Lägg till ett fält “Filtrera på plats”

    6. visa bara de aktiviteter som matchar texten i filtret

    7. Snyggare lista med CSS

    8. Gör listan tydligare med t.ex.:
      • annan bakgrundsfärg
      • lite padding/marginaler
      • större radavstånd

    Extra nivå (för den som vill leka mer)¶

    Om du tycker det är kul och orkar mera:

    • Testa att hämta startdatan från ett API (se 2.3) och sedan lägga till lokalt
    • Lägg till en knapp för att rensa listan (t.ex. sätt activities = [] och kör renderActivities())
    • Dela upp din JavaScript-kod i flera filer eller funktioner om du känner att det börjar bli långt

    Kom ihåg: det är helt okej att bara titta på hur det skulle kunna se ut – du behöver inte få allt perfekt.

    Vad du kan visa upp¶

    Om du vill ha feedback kan du:

    • dela länken till din aktivitetslista-sida i kursens chatt
    • skriva kort:
    • vad din lista föreställer (t.ex. “FVK-aktiviteter”, “min vecka”)
    • vad du vill förbättra

    Detta hjälper kursledaren att ge konkretare tips.

    Fortsätta på egen hand¶

    Efter Paket 2 kan du t.ex.:

    • bygga fler "små appar" på samma sätt (inköpslista, filmtips, träningslogg …)
    • experimentera med fler JavaScript-funktioner (sortering, olika vyer)
    • börja kika på hur ramverk som t.ex. SvelteKit gör liknande saker, fast mer strukturerat

    I Paket 3 kommer du få se hur moderna ramverk kan hjälpa dig att bygga vidare på samma idéer på ett mer organiserat sätt.
