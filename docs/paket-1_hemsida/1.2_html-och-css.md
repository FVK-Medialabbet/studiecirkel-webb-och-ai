# 1.2 HTML och CSS

## Mål

Efter den här modulen ska du kunna:

- förstå hur HTML strukturerar innehållet på en webbsida
- använda vanliga HTML-element (rubriker, text, länkar, listor, bilder)
- koppla in en CSS-fil
- ändra färger, typsnitt och enkel layout med CSS

## Innehåll

1. **HTML – strukturen**
   - grundstruktur: `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`
   - rubriker (`<h1>`–`<h3>`), stycken (`<p>`), länkar (`<a>`), listor (`<ul>`, `<li>`), bilder (`<img>`)
2. **Skapa en enkel startsida**
   - titel
   - en huvudrubrik
   - ett par stycken text
   - en länk
   - en bild
3. **CSS – utseendet**
   - koppla in en CSS-fil med `<link>` i `<head>`
   - färger (`color`, `background-color`)
   - typsnitt (`font-family`, `font-size`)
   - marginaler och mellanrum (`margin`, `padding`)
4. **Enkel layout**
   - block och inline-element
   - introduktion till `display: flex` för att placera saker bredvid varandra

## Förslag på självstudier

- Bygg en startsida med:
  - minst en rubrik
  - minst två stycken text
  - en lista (t.ex. favoritintressen)
  - en bild
- Prova tre olika färgteman och typsnitt – välj ett du gillar och spara det.
- Lägg till en enkel meny överst med länkar som *än så länge* bara pekar på samma sida.
