## Mål

Efter den här modulen ska du:

- veta var JavaScript hör hemma i en webbsida
- kunna koppla en JS-fil till din sida
- kunna lägga till en enkel interaktiv funktion (t.ex. meny som fälls ut)

## Innehåll

1. **Varför JavaScript?**
   - göra sidan mer ”levande”
   - reagera på klick och andra händelser
2. **Koppla in en JS-fil**
   - skapa t.ex. `script.js`
   - länka in filen i slutet av `<body>`
3. **Exempel: meny som kan fällas ut**
   - lägga till en knapp (”Meny” eller ikon)
   - JavaScript som:
     - hittar menyn i HTML (`document.querySelector`)
     - byter CSS-klass när man klickar på knappen
   - styra om menyn ska vara synlig eller dold med CSS-klasser
4. **Små animationer**
   - enkel `transition` i CSS för att menyn ska fällas ut mjukt

## Förslag på självstudier

- Lägg till minst en interaktiv detalj på din sida, t.ex.:
  - meny som fälls ut/in
  - en knapp som visar eller gömmer extra text (”Läs mer”)
  - en knapp som byter bakgrundsfärg eller tema
- Skriv kort (för dig själv eller i Zulip) vad du tror JavaScript kan användas till mer i framtida moduler.
