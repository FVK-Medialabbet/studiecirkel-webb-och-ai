# 1.1 Git, GitHub och GitHub Pages

## Mål

Efter den här modulen ska du kunna:

- skapa ett konto på GitHub
- skapa ett nytt repository för din hemsida
- göra enklare ändringar och spara dem med Git (commit/push)
- aktivera GitHub Pages och hitta adressen till din hemsida

## Förberedelser

- GitHub-konto (skapa på [github.com](https://github.com) om du inte redan har ett)
- en enkel textredigerare eller VS Code installerat
- grundläggande datorvana (spara filer, hitta mappar)

## Innehåll

1. **Varför Git och GitHub?**
   - versionshantering: spara historik, ångra misstag
   - samarbete: flera kan jobba på samma projekt
2. **Skapa repo för hemsidan**
   - nytt repository på GitHub
   - lägga till en enkel `index.html`
3. **Spara ändringar med Git**
   - `commit` = spara en version
   - `push` = skicka upp till GitHub
4. **Aktivera GitHub Pages**
   - slå på Pages i repo-inställningar
   - välja branch och ev. mapp
   - hitta webbadressen till sidan

## Förslag på självstudier

- Gör minst två små ändringar (t.ex. ändra text i `index.html`) och:
  1. spara med `commit`
  2. skicka upp med `push`
  3. kontrollera att GitHub Pages-sidan uppdaterats
- Testa att titta på commit-historiken i GitHub och se vilka ändringar du gjort.
